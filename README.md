PSProgressHUD
=============

iOS Progress HUD that animates a custom set of images 

### How to use

1. Copy PSProgressHUD.h and PSProgressHUD.m into your Xcode project.
2. Copy the HUD images to your `images.xcassets` 

To show the HUD: 

`[PSProgressHUD show] `

To dismiss the HUD:

`[PSProgressHUD dismiss] `
